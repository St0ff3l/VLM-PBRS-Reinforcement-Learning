## 二、研究背景与动机

### 2.1 问题陈述

强化学习（RL）在复杂任务中展现出强大能力，但面临**样本效率低下**的核心挑战：
- 像MountainCar这种需要“先向左蓄力，再向右冲锋”的长时序稀疏奖励任务，RL智能体如果仅靠盲目探索，极易陷入策略坍缩（如一直给出固定无效动作），导致训练停滞、奖励全负。
- 传统奖励塑形（Reward Shaping）高度依赖专业领域知识进行硬编码，难以零成本泛化到其他环境。

### 2.2 研究动机：为什么要引入 VLM？

视觉语言模型（VLM，如LLaVA）具备**开放世界语义理解与零样本推理能力**。
当我们给定自然语言先验（例如：“要到达山顶，小车必须先向左移动积累动量，然后加速向右坡冲刺”），VLM能够：
- 直接“看”环境画面，对比智能体当前动作与语言先验的匹配度。
- 将原本极度稀疏的环境奖励，转化为稠密、富有语义指导意义的数值奖励（0~1的匹配分数）。
- 通过这种**基于自然语言指令的视觉奖励塑形**，引导智能体走出早期局部极小值，解决策略坍缩，从而显著提升RL的样本效率。

### 2.3 初步探索

### 4.2 核心算法设计

#### （1）自然语言先验的语义奖励机制

不再硬编码规则！我们将任务逻辑翻译成给 VLM 的指令。VLM 输入：当前视觉帧 + 语言指令。
VLM 直接输出：0.0~1.0 之间的行为匹配度分数（Φ, Potential）。

```text
Prompt:
"Analyze the image of the reinforcement learning environment. 
Evaluate how closely the car's current state aligns with the strategic prior: 
'First, heavily move to the left slope to build mechanical momentum, 
then violently accelerate to the right slope to reach the goal flag.'
Output ONLY a float between 0.0 (completely failing/stuck) and 1.0 (perfect execution and reaching goal)."
```

基于输出的分数实现 PBRS：
```python
# PBRS奖励塑形（保证最优策略不变，且防止反复横跳刷分）
shaping_reward = γ * Φ(s') - Φ(s)
```

#### （2）自适应探索强化（防坍缩）

鉴于 PPO 在稀疏奖励下易坍缩为单一动作失效：
- **提高熵系数 (Entropy Coefficient)**：从常规的 0.00~0.01 提升至 0.05~0.1，强制网络探索动作空间。
- **环境奖励正向化**：给予完成任务正奖励（或者在包裹器里修正极度负偏的 reward），避免智能体完全摆烂。

#### （3）双门控节流（缓解多模态高延迟）

为解决在线 VLM 推理延迟导致的训练停滞问题，采用“双门控（dual-gating）”策略：
1. **视觉新颖度门控**：仅当 `MSE(帧_t, 帧_上一缓存) > τ_mse` 时，认为语义状态发生变化。
2. **时间节流门控**：仅当 `t mod N = 0` 时允许真实调用 VLM。

势函数更新形式：
```
Φ_{t+1} = VLM(s_{t+1}),   if Trigger == 1
Φ_{t+1} = Φ_t,            if Trigger == 0
```
